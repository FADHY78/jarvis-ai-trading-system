âš ï¸ TROUBLESHOOTING

If Telegram doesn't work:
1. Check VPN is connected
2. Restart backend: Ctrl+C, then npm run backend
3. Verify .env.local has correct token/chat ID
4. Click START button in Telegram bot chat

If Deriv doesn't connect:
1. Check internet connection
2. Verify token is exactly 15 characters
3. Check token hasn't expired
4. Try demo mode (empty token)

If Port already in use:
1. Close other instances
2. Or use: netstat -ano | findstr :3001
3. Then: taskkill /PID <PID> /F

If npm install fails:
1. Delete node_modules folder
2. Delete package-lock.json
3. Run: npm install again
